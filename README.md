#Five recents crime reported
-------------------------------------------------------------------------------

It's an application that displays five recents crime reported in New York city by typing the zip code.

#Technologies Used
-------------------------------------------------------------------------------

- HTML
- CSS
- JavaScript
- jQuery
- Hosted on Vercel
- AJAX used to receive data from ana external source NYPDapi.
- NYPD API
